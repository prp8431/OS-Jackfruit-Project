# OS Jackfruit - Container Runtime Simulation

## Description
This project simulates a container runtime system using fork() and exec().

## Features
- Start and stop containers
- Logging system
- Command-line execution
- Container activity tracking

## Concepts Used
- fork()
- exec()
- kill()
